## Overview
The **Energy-Efficient Smart Wheel Hoe** is an embedded systems project aimed at developing a low-cost, intelligent, and energy-efficient farming aid for small and medium-scale agriculture.  
The system focuses on **motorized soil operation, real-time control, and user interaction**, using an STM32 microcontroller as the core controller.

## Objectives
- Reduce manual labor in farming operations
- Improve energy efficiency using embedded control
- Design a modular and scalable embedded firmware architecture
- Provide real-time system monitoring via an HMI

---

## System Architecture
- **Controller:** STM32 (ARM Cortex-M series)
- **Actuators:** DC motors, servo motors
- **User Interface:** TouchGFX-based HMI
- **Communication:** UART
- **Power System:** Battery-based electric drive

---

## Features Implemented
- STM32-based embedded firmware framework
- DC motor control logic
- Servo motor actuation
- UART-based communication
- Modular driver-oriented firmware structure
- TouchGFX HMI design for:
  - Battery status monitoring
  - Speed display
  - Operating mode selection
